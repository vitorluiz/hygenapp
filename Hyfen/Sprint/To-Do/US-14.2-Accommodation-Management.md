# [US-14.2] Gestão Detalhada de Tipos de Acomodação

**Épico:** 8 - Gestão Detalhada de Acomodações (Refinamento)

**User Story:**
> **Como** um Proprietário,
> **eu quero** poder criar e gerenciar diferentes **tipos** de acomodações (Quarto Standard, Suíte Master, Loft, etc.) com galerias de fotos e descrições próprias,
> **para que** eu possa apresentar meu inventário de forma clara e atraente para os hóspedes.

### Critérios de Aceite:

1.  ✅ A interface de gestão de acomodações no dashboard deve ser clara e permitir a criação de múltiplos tipos de quartos.
2.  ✅ Para cada tipo de acomodação, o proprietário pode adicionar um **nome**, uma **descrição detalhada**, uma lista de **comodidades específicas** (ex: "Banheira de Hidromassagem", "Vista para o Mar") e o **preço base**.
3.  ✅ O proprietário pode criar uma **galeria de fotos específica para cada tipo de acomodação**.
4.  ✅ Na Landing Page pública, as acomodações devem ser listadas em seções separadas e claras, cada uma com sua própria galeria de fotos e detalhes.

# [US-13.2] Funcionalidade de Busca e Filtro (MVP)

**Épico:** 13 - Portal de Descoberta Hyfen

**User Story:**
> **Como** um Hóspede,
> **na** página de descoberta (`/explore`), **eu quero** poder buscar por cidade ou nome da propriedade,
> **para que** eu possa encontrar rapidamente o que estou procurando.

### Critérios de Aceite:

1.  ✅ Um campo de busca de texto deve ser adicionado à página `/explore`.
2.  ✅ O endpoint da API `GET /api/v0/public/properties/` deve ser modificado para aceitar um parâmetro de query `?search=...`.
3.  ✅ A lógica de busca no backend (Django) deve filtrar as propriedades cujo nome ou cidade contenham o termo pesquisado.
4.  ✅ A interface no frontend deve refazer a chamada à API com o termo de busca e atualizar o grid de resultados dinamicamente, sem precisar recarregar a página inteira.
